\section{Principales aportaciones}
El desarrollo de este proyecto concurre en aportaciones en dos entornos: el social y el técnico.

\subsection{Social}
En el entorno social se distinguen como aportaciones:

\begin{itemize}
    \item La existencia de este espacio promueve la difusión del propio instrumento, ayudando a preservar su legado cultural y a aumentar su popularidad entre nuevas generaciones y diferentes regiones geográficas.

    \item La aplicación web ofrece a los usuarios recursos para el aprendizaje y práctica del timple, facilitando la accesibilidad al instrumento y fomentando su estudio.

    \item La aplicación web fomenta la creación de contenido musical para el timple, facilitando a los usuarios la publicación y compartición de canciones.

    \item Aunque ya existen comunidades de timplistas, este espacio digital puede servir como punto de encuentro para todos aquellos interesados en el instrumento, promoviendo la interacción y colaboración entre sus miembros.
\end{itemize}

\subsection{Técnico}

Entre las aportaciones en el entorno técnico encontramos:

\begin{itemize}
    \item El desarrollo de un espacio digital donde el timple sea el protagonista.
    \item La creación de utilidades adaptadas a las singularidades técnicas y culturales del instrumento.
    \item El diseño e implementación de un algoritmo de cálculo de acordes para instrumentos de cuerda.
\end{itemize}

Dada la escasez de recursos digitales la creación de esta aplicación web representa un importante desafío, siendo las unicas referencias aquellas soluciones existentes diseñadas para otros instrumentos.
